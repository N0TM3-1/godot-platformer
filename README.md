# Godot Platformer

A simple 2D platformer made for learning the basics of Godot.

## About The Project

A small learning project to explore Godot's fundamentals:
- Move a player character around
- Collect coins scattered in the level
- Handle basic physics and input
- Organize code into scenes and scripts

## How to Play

- Arrow keys or WASD to move
- Walk into coins to collect them

## What's Inside

- `player.gd` - Player movement and jumping
- `coin.gd` - Coin collection logic
- `area_2d.gd` - Area detection
- `manager.gd` - Game management
- `main.tscn` - Main level scene
- `coin.tscn` - Coin prefab
